==== 📋 每日記錄功能說明 ====

【功能介紹】
系統在輪詢取得機台數據後，會自動檢查並更新每日記錄

【核心邏輯】
1. 每次輪詢獲得投幣數和出貨次數
2. 與今天的記錄比對
3. 如果相同 → 不更新 (ℹ️ unchanged)
4. 如果不同 → 更新記錄 (🔄 updated)
5. 如果今天還沒有記錄 → 新建記錄 (✅ created)

【記錄結構】
```javascript
window.dailyRecords[recordKey] = {
    devid: 1,                              // 機台 ID
    date: "2025-10-23",                    // 日期 (YYYY-MM-DD)
    plays: 135,                            // 當前投幣數
    payouts: 9,                            // 當前出貨次數
    createdAt: "2025/10/23 上午 11:52:59", // 創建時間
    updatedAt: "2025/10/23 上午 11:55:30", // 最後更新時間
    updateCount: 3,                        // 今天更新次數
    history: [
        {timestamp: "上午 11:52:59", plays: 135, payouts: 9},
        {timestamp: "上午 11:53:15", plays: 137, payouts: 10},
        {timestamp: "上午 11:55:30", plays: 140, payouts: 11}
    ]
}
```

【查詢方式】
1. 在 Console 查看所有記錄:
   console.log(window.dailyRecords);

2. 查看特定機台的今天記錄:
   today = new Date().toISOString().split('T')[0];
   record = window.dailyRecords[`devid_1_${today}`];
   console.log(record);

3. 查看特定機台的更新歷史:
   record.history.forEach(h => console.log(h));

【Console 日誌示例】
輪詢時會看到:

✅ devid 1 數據已儲存: {plays: 135, payouts: 9, ...}
📝 檢查並更新每日記錄 - devid: 1, plays: 135, payouts: 9
✅ 新建今日記錄 - devid: 1, 日期: 2025-10-23, plays: 135, payouts: 9
📋 每日記錄更新結果 (created): {devid: 1, date: "2025-10-23", ...}

再次輪詢:
✅ devid 1 數據已儲存: {plays: 137, payouts: 10, ...}
📝 檢查並更新每日記錄 - devid: 1, plays: 137, payouts: 10
🔄 數據有變化 - 舊: plays=135, payouts=9, 新: plays=137, payouts=10
✅ 已更新今日記錄 - devid: 1, plays: 137, payouts: 10, 更新次數: 2
📋 每日記錄更新結果 (updated): {devid: 1, date: "2025-10-23", ...}

再次輪詢 (無變化):
✅ devid 1 數據已儲存: {plays: 137, payouts: 10, ...}
📝 檢查並更新每日記錄 - devid: 1, plays: 137, payouts: 10
ℹ️ 數據未變化 - devid: 1, plays: 137, payouts: 10 (維持不變)
📋 每日記錄更新結果 (unchanged): {devid: 1, date: "2025-10-23", ...}

【返回值】
updateDailyRecord() 返回:
{
    status: "created" | "updated" | "unchanged",
    record: { 完整的記錄對象 }
}

【關鍵特性】
✅ 自動按日期分組 (YYYY-MM-DD)
✅ 自動按 devid 區分 (支持多個機台)
✅ 記錄完整的更新歷史
✅ 統計今天的更新次數
✅ 記錄首次創建和最後更新時間

【後續可實現】
- 保存到本地存儲 (localStorage) 或 IndexedDB
- 导出到 CSV/Excel
- 圖表展示每日變化趨勢
- 設定告警 (如果變化超過某個閾值)
- 統計每日/每週/每月的增量
