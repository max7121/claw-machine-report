📚 使用步驟：

  1. 確認上方 MQTT 連線狀態為「✅ 已連接」
  2. 輸入要讀取的機台 MAC 位址（預設已填入測試位址）
  3. 選擇天車 ID（左天車或右天車）
  4. 點擊「📤 發送讀取命令」按鈕
  5. 系統會發送 JSON 格式命令到主題 mqtt/s
  6. 機台收到命令後會回傳 65 位元組的參數資料
  7. 回傳資料會自動顯示在下方的命令記錄中
🔍 點擊查看命令格式說明

發送格式 (JSON):

{
  "cmd": "readdata",
  "mac": "083A8DE1EC88",
  "floor": "1",
  "devid": "1",
  "taskid": "時間戳"
}

回應格式 (JSON):

{
  "cmd": "re_readdata",
  "mac": "ClientID",
  "floor": 1,
  "devid": 1,
  "parm": "65位元組HEX資料",
  "taskid": "相同的時間戳"
}

💡 parm 欄位包含 Byte8~Byte72 的所有機台參數（投幣設定、遊戲參數、電壓、統計等）
